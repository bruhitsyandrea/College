Lab 3: Looping in RISC-V Assembly

DESCRIPTION
This lab take an input number from the user to form a pattern of hollow right-angle triangle. If the number is less than or equal to zero, the program will ask the user to re-enter until the input number is greater than zero.

FILES
-
Lab3.asm
This file includes the assembly code of the lab.

INSTRUCTIONS
This program is intended to be run using the RISCV Assembler and Runtime Simulator
(RARS). Enter the test case as a program argument and run using RARS.
